TrainingPartner

React | Ruby on Rails | PostgreSQL | SCSS

Brazillian Jiujitsu is a fun and fast-growing sport, but it can be a bit overwhelming for newcomers. I built TrainingPartner to help people new to the game keep track of techniques as they learn them, and log information on training sessions to see their progress as the weeks go by.

eKnow

React | CSS

Oblique Solutions, co-created by Brian Eno (get it?), is a card set with vague but helpful advice designed to help musicians overcome challenges during writing and recording. Leveraging the oblique solutions API, I built an app that pairs text from a random card with random imagery, and allows the user to muse on how the advice applies to their own situation. It offers users a chance to have a one-way conversation with yourself to find perspective, a little like rubber-ducky coding.
